# XPress

## About
ASPS One is the reference case of a web app. The fictitious app, XPress, is a blogging site.

## Assessed Features
- Self-configuring:
    - HTTP payloads
    - API requests
- Self-healing:
    - Network errors
    - Detecting DDoS
- Self-optimising:
    - Algorithms used
    - Precaching images
- Self-protecting:
    - Passwords
    - Rate limits
    - Encryption of data (JWTs)
